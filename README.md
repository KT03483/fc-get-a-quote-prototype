# FC Get A Quote — Earn the Form

Lo-fi clickthrough prototype for the Flight Centre **Get A Quote** redesign.

**Direction B: Earn the Form** — leads with consultant trust before asking for any form fields, reframing the experience from filling out a database to writing to a real person.

## Screens

1. **Form** — Hero/trust block · Intent signal · Trip details · Contact details · Sticky sidebar
2. **Confirmation** — Post-submission state with consultant follow-up messaging

## Design system

Styled with TakeFlight design system tokens — color, spacing, radius, and typography.

## Usage

Open `earn-the-form-prototype.html` directly in a browser. No build step required.
